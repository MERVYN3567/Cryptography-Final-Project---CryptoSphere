# Cryptography-Final-Project---CryptoSphere
CryptoSphere is a high-performance, reactive web application built as a specialized training simulator for secondary school students. By moving from a Java backend to a full-stack TypeScript/React approach on Vite, the app provides instantaneous feedback, which is crucial for maintaining the engagement of 13–17-year-olds.

Link to the Website: https://cryptosphere-project.vercel.app/
